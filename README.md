# Blog Project
静态博客网站，使用Vuepress构建工具生成，主要内容为学习笔记，在使用一些工具、框架或者感兴趣的东西都会记录下来。

## 启动
::: tip
npm run docs:dev
:::

## 生成静态资源
::: tip
npm run docs:build

## git 路径
$ git push https://github.com/fly-liu/blog-project.git master
